HOW TO USE:

1. Launch GUI:
   ▶ python gui.py

2. Set Target Company:
   ▶ Enter company name → Click "Set Company"

3. Start Monitoring:
   ▶ Click "▶ Start Crawl"
   ▶ Watch progress in Logs tab
   ▶ Report auto-generates (2-5 seconds)
   ▶ Findings auto-load in table

4. Review Results:
   Option A: Check Dashboard tab for overview
   Option B: View All Findings tab for details
   Option C: Open PDF Report for comprehensive analysis

5. Take Action:
   ▶ Filter by risk level
   ▶ Right-click for details
   ▶ Export to CSV
   ▶ Review mitigation advice

RISK INDICATORS:
   
   🔴 CRITICAL (Score >= 85)    →  Immediate action required
   🟠 HIGH      (70-84)         →  Address within 24 hours
   🟡 MEDIUM    (50-69)         →  Address within 1 week
   🟢 LOW       (< 50)          →  Monitor and review
